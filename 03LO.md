## [Largest Odd Number in a String. & https://leetcode.com/problems/largest-odd-number-in-string/description/]

**Rating:** (Easy)
**Topic:** (String, Stack)
**Pattern:** (Simple Strings & For loop)

### 🧠 The "Aha!" Moment (The Logic)

_Given a string s, representing a large integer, the task is to return the largest-valued odd integer (as a string) that is a substring of the given string s._
_The number returned should not have leading zero's. But the given input string may have leading zero.._

- Focus on finding the longest valid odd number starting from the original string.
- An odd number must end with an odd digit, so we look for the last odd digit when scanning from the end.
- Leading zeroes don’t add value to the number, so we aim to remove them for a cleaner representation.
- Once the endpoint (last odd digit) is determined, we identify the starting point by skipping any leading zeroes before it.
- Extract the portion between these two positions, this gives the largest possible odd integer from the string."\_
